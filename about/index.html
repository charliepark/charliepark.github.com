This Tumblelog

I created the framework for this tumblelog (using Eleventy) because I wanted to get to know the software a bit better, wanted to play around with Netlify, and wanted to post more on my site. It was feeling kind of staid, and it felt like things had to be just right to merit posting. That’s silly. So! A tumblelog.

If you’re interested in an Eleventy-based tumblelog like this one, you’re welcome to check out the code, though I need to polish it up a bit before it works out of the box.
